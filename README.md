# vue-guide-instance
# 官网实例

# -----------------------------
# vuecli2是vue cli2介绍，
# vuevite式vue cli3介绍


# 安装vue-cli

```js
yarn global add @vue/cli

npm install -g @vue/cli
```

# 创建项目
```js
// create
npm create 项目名

// 可视化
npm ui
```

也可以使用vite创建
```js
npm init vite-app 项目名
cd 项目名
npm install
npm run dev
```
